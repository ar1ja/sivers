help moderate our directory!

nownownow.com is a community-run directory of /now pages, which includes some older pages that haven't been updated in years.

If you have some spare time, please help us by using our checking tool and follow these steps:

  1. Visit a site and find the date when it was last updated.
  2. Copy that date text (for example, "as of july 2025") from the site and paste it into the form on the moderation page, so our automated system can recognize it in the future
  3. Select the last-updated date so that nownownow.com can sort pages by their latest update
  4. Choose whether the site is gone, very old, no date shown, or if everything looks good

The tool will always show you the next site to check, and you can stop at any time.

Thank you for your help!
